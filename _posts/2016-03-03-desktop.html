Lesson Planner

Built a C# Windows 8 application that lets teachers collaborate on their lesson plans. It was built with C#, PHP, MySQL, and JSON. The C# makes a call to a PHP file, which generates JSON from the MySQL Database, which then is used to display the data in the app. Check it out.
